# Carpeta de Flyers de Eventos

**NOTA**: Ya no necesitas subir archivos aquí. Ahora usamos **PostImages.org** para alojar las imágenes de los flyers.

## Cómo Subir tu Flyer de Evento:

1. **Visita PostImages.org**: Abre [https://postimages.org/](https://postimages.org/) en tu navegador
2. **Sube tu imagen**: Haz clic en "Choose images" y selecciona tu flyer
3. **Copia el link directo**: Una vez subida, copia el link que dice "Direct link" o termina en `.jpg`, `.png`, etc.
4. **Pega en el admin**: Usa ese link en el campo "URL del Flyer del Evento" en el panel de administración

## Especificaciones de Imagen Recomendadas:
- **Formato**: JPG, PNG, o WebP
- **Dimensiones**: 1080x1350px (vertical) o 1200x628px (horizontal)
- **Tamaño de Archivo**: Mantén bajo 2MB para carga óptima

## Ejemplo de Link:
```
https://i.postimg.cc/abcd1234/kuff-miami-2025.jpg
```

¡Es así de fácil! No necesitas acceso FTP ni subir archivos manualmente.
