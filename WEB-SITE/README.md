# KUFF - Official Website

Professional website for KUFF, International DJ and Producer.

## Features

- **Modern Design**: Dark theme with cyan accent colors matching the brand identity
- **Fully Responsive**: Optimized for all devices (desktop, tablet, mobile)
- **Smooth Animations**: Scroll animations, hover effects, and interactive elements
- **Performance Optimized**: Lightweight and fast loading
- **SEO Friendly**: Proper meta tags and semantic HTML

## Sections

1. **Hero Section**: Eye-catching landing with animated logo
2. **About**: Biography and career highlights
3. **Supported By**: Artists who support KUFF's music
4. **Music & Platforms**: Links to streaming platforms and labels
5. **World Tour**: Countries and regions where KUFF has performed
6. **Contact**: Booking information and social media links

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Grid/Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts (Montserrat)

## How to Use

1. Open `index.html` in a web browser
2. Or deploy to any web hosting service (Netlify, Vercel, GitHub Pages, etc.)

## Customization

### Colors
Edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #00d9ff;
    --secondary-color: #0099cc;
    --accent-color: #00ffff;
    --dark-bg: #000000;
}
```

### Content
Update the content directly in `index.html` or add your own sections.

### Social Links
Add your social media links in the Contact section by replacing the `#` in the href attributes.
